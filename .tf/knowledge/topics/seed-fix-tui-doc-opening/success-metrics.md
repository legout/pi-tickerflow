# Success Metrics

- [ ] Pressing `o` opens the first available document for the selected topic
- [ ] Pressing `1` opens the overview document (if available)
- [ ] Pressing `2` opens the sources document (if available)
- [ ] Pressing `3` opens the plan document (if available)
- [ ] Pressing `4` opens the backlog document (if available)
- [ ] Documents open correctly in `$PAGER` or `$EDITOR`
- [ ] TUI resumes correctly after closing the pager/editor
- [ ] Appropriate notifications shown when document doesn't exist or no topic selected
