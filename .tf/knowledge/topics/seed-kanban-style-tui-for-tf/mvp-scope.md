# MVP Scope

## In Scope

- A `tf` subcommand to launch the TUI (name TBD).
- List/board view showing ticket columns derived from status + deps (ready/blocked).
- Search/filter tickets by status + tag + substring match.
- Ticket detail panel (read-only).
- Basic topic list for seeds/spikes/plans with open/view.

## Out of Scope (MVP)

- Editing ticket content from the UI.
- Drag-and-drop.
- Full markdown rendering fidelity.
- Synchronizing across machines.
