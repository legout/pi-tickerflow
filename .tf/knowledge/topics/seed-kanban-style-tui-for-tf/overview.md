# seed-kanban-style-tui-for-tf

Build a Kanban-style terminal UI (Python) for Ticketflow so users can browse tickets and knowledge topics with fast search/filtering.
The UI should make it easy to see “what’s next”, drill into ticket details, and navigate seed/spike/plan/backlog docs.
